Summary

This project explores authorship identification using essays by four contemporary essayists: David Foster Wallace, Joan Didion, Zadie Smith, and John Jeremiah Sullivan. Despite stylistic similarities, the goal was to determine whether computational metrics—such as part-of-speech distributions, lexical diversity, and unique word counts—can be used to distinguish authorship.

Methodology

I collected essays by each author, extracting and cleaning the text. Each essay was split into chunks of three paragraphs and filtered to retain segments with 100–200 words. To ensure a balanced comparison, six chunks were randomly selected per author.

Part-of-speech tagging was done using the nltk library with the universal tagset. For each chunk, I calculated:

  • POS tag distributions
  • Unique word count
  • Lexical diversity: (unique tokens / total tokens) * 100

Results

POS Tag Trends

Notable patterns:

  • Zadie Smith used the most adjectives (15 on average).
  • Joan Didion had the lowest adverb use (8.8).
  • Sullivan and Smith used more pronouns—potentially indicating a more personal narrative style.
  • Didion and Wallace had higher noun counts per chunk.
Lexical Measures

Average Unique Word Count per Chunk

  • David Foster Wallace: 71.5
  • John Jeremiah Sullivan: 71.3
  • Joan Didion: 65.7
  • Zadie Smith: 63.8

Average Lexical Diversity

  • David Foster Wallace: 85.79%
  • John Jeremiah Sullivan: 84.57%
  • Zadie Smith: 83.22%
  • Joan Didion: 81.87%

Wallace and Sullivan exhibited slightly higher lexical diversity, with Didion and Smith clustering slightly lower. These subtle distinctions suggest that lexical features may offer some predictive power, though the dataset is small.

Process

Data Collection & Sampling
  • Custom Python scripts were used to extract and clean text.
  • Each text was tokenized, POS-tagged, and filtered based on word count.
  • Random chunk sampling ensured balanced representation per author.
Analysis Pipeline
  • nltk for tokenization, POS tagging, and stopword filtering
  • pandas and matplotlib for data aggregation and visualization
  • Calculated and visualized metrics: POS frequencies, unique word counts, lexical diversity

Tools & Technologies

  • Languages: Python
  • Libraries: NLTK, pandas, matplotlib, scikit-learn
  • Data Format: Plain text, organized by author
